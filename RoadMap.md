# Project Roadmap: Phael Flor Catalog App

| Phase | Task | Duration |
| :--- | :--- | :--- |
| **Phase 1: Setup** | Supabase Project creation & Database Schema design. | Week 1 |
| **Phase 2: Assets** | Uploading product images to Storage & filling JSON data. | Week 1-2 |
| **Phase 3: UI Dev** | Building the Home Screen & Brand Filters in Flutter. | Week 2 |
| **Phase 4: Logic** | Implementing the Advanced Detail Page with dynamic tags. | Week 3 |
| **Phase 5: Optimization** | Implementing Search and Image Caching. | Week 4 |
| **Phase 6: QA** | Testing on Android/iOS and performance tuning. | Week 5 |
| **Phase 7: Launch** | Deployment to Play Store / App Store. | Week 6 |

## Future Evolution (Phase 2)
* Multi-language support (FR/EN).
* PDF Export: Generate a technical data sheet for professional buyers.
* Offline Mode: Sync data for use without internet.
